🗳️ Online Voting System (Java Swing)

📌 Project Overview

This is a desktop-based Online Voting System developed using Java Swing.
The application allows users to register, login, and cast their vote securely with a modern UI.

---

🚀 Features

- 👤 User Registration & Login
- 🔐 Input Validation (No empty fields, no duplicate usernames)
- ⌨️ Enter Key Support for form submission
- 🗳️ One User = One Vote
- 🖼️ Logo-based Voting Interface
- 📊 Result Display with Winner & Party Logos
- 🎨 Modern Dark UI with hover effects

---

🧱 Technologies Used

- Java (Core)
- Java Swing (GUI)
- AWT (Layouts & Events)

▶️ How to Run

1. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
2. Make sure all image files are in the same folder as "VotingGUI.java"
3. Compile and run:

javac VotingGUI.java
java VotingGUI

🔐 Validation Rules

- Username & Password cannot be empty
- Duplicate usernames are not allowed
- A user can vote only once
